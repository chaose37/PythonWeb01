MVC : MTV

다른 프레임워크와 달리 장고는 MTV 패턴의 방식을 사용한다.

View는 MVC의 Controller Templet은 View 와 비슷하게 사용된다.

장고에서 url 매핑설정은 urls.py 파일에서 설정이 가능하다.
매핑된 객체에서 httpResponse 객체를 통해 html 문을 작성하여 페이지에 값을 쓸수있다
또한 Render 객체를 import 하여 이미작성된 html 페이지로 이동시켜줄수도 있다
이때 Render 객체의 파라미터를 이용해 모델의 값을 포함해서 넘기는것이 가능하다
장고에서의 DB 설정은 Settings.py 파일에서 Database 객체를 찾아서 등록한다
모델추가는 project/models.py 파일에서 추가한다.
모델을 추가한 후에는 admin.py 에 등록해야한다.
등록후에 어드민페이지를 통해 데이터를 추가하려 하면 테이블이 존재하지않는다고 에러가 난다
서버를 종료한 후 터미널을 켜고 python manage.py makemigration 실행하면 DB 생성쿼리문을 준비한다
python manage.py migrate 명령을 실행하여 쿼리문을 실행하면 자동으로 테이블을 추가한다.
템플릿으로 모델객체를 전송할 수 있다. 사용방법은 JSP와 EL 태그와 유사하다
for문 if문등 파이썬의 문법을 사용가능하며 객체는 객체명.속성명으로 속성에 접근 가능하다.
